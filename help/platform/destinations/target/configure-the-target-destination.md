# Configuración del destino de Adobe Target en Real-time Customer Data Platform

Obtenga información sobre cómo configurar el destino de Adobe Target en Real-time Customer Data Platform para que empiece a enviar segmentos y atributos de perfil de Real-Time CDP a Target. Para obtener más información, consulte la [Documentación de conexión de Adobe Target](https://experienceleague.adobe.com/docs/experience-platform/destinations/catalog/personalization/adobe-target-connection.html?lang=es).

>[!IMPORTANT]
>
>Con la versión beta del conector de destino Adobe Target V2 mejorado, es posible que vea dos tarjetas Adobe Target en el catálogo de destinos. El conector de destino Adobe Target V2 está en versión beta y solo está disponible para un número determinado de clientes.
