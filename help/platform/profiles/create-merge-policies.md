# Crear políticas de combinación

Este vídeo muestra cómo crear políticas de combinación en Adobe Experience Platform. Las políticas de combinación son las reglas que utiliza Platform para determinar qué datos se utilizarán y priorizarán al combinar conjuntos de datos de fuentes diferentes, con el fin de crear perfiles de clientes.

>[!VIDEO](https://video.tv.adobe.com/v/330433?quality=12&learn=on)

Para obtener más información, visite [documentación de políticas de combinación](https://experienceleague.adobe.com/docs/experience-platform/profile/merge-policies/overview.html).
