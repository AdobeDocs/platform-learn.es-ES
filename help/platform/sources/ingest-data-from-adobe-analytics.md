# Ingesta de datos mediante el conector de origen de Adobe Analytics

El conector de origen de Adobe Analytics le permite transmitir, asignar y filtrar fácilmente datos de Adobe Analytics al Perfil del cliente en tiempo real y al lago de datos de Experience de Adobe Experience Platform. Aprenda a utilizar las funciones de preparación de datos para crear campos semánticos y calculados para mejorar la experiencia del usuario en las aplicaciones del Generador de segmentos y la Plataforma . Este conector admite grupos de informes multiregionales. Para obtener documentación del producto más detallada, consulte [Creación de un conector de origen de Adobe Analytics en la interfaz de usuario](https://experienceleague.adobe.com/docs/experience-platform/sources/ui-tutorials/create/adobe-applications/analytics.html?lang=es).
