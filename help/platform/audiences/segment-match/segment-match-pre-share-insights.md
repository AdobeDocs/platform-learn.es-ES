# Perspectivas previas al uso compartido de coincidencias de segmentos

Cuando decida con qué socio estratégico va a compartir datos, es importante saber cómo coinciden sus clientes, de modo que sepa cuán útil será ese uso compartido de datos. La coincidencia de segmentos le permite ver la superposición con posibles socios de datos antes de compartir datos. Para obtener más información, consulte la [documentación](https://experienceleague.adobe.com/en/docs/experience-platform/segmentation/ui/segment-match/overview){target="_blank"}.

>[!IMPORTANT]
>
>Adobe introdujo la coincidencia de segmentos en 2021 para que los clientes colaboren e intercambien audiencias. A principios de 2025, Adobe presentó Real-Time CDP Collaboration, el enfoque a más largo plazo para satisfacer este caso de uso.
>
> * Para clientes de Estados Unidos, Canadá, EMEA, Australia y Nueva Zelanda: Adobe recomienda a los clientes de Real-Time CDP Prime y Ultimate realizar la transición de casos prácticos de colaboración de datos de Coincidencia de segmentos a Real-Time CDP Collaboration. Consulta la [documentación](https://experienceleague.adobe.com/en/docs/real-time-cdp-collaboration/using/home){target="_blank"} y la [guía de inicio rápido](https://experienceleague.adobe.com/en/docs/real-time-cdp-collaboration/using/quick-start-guide){target="_blank"} para Real-Time CDP Collaboration y ponte en contacto con el equipo de tu cuenta de Adobe para obtener más información.
> * Para clientes de todas las demás regiones geográficas: La coincidencia de segmentos es la opción recomendada hasta que se publique Real-Time CDP Collaboration en esas regiones geográficas en 2026.
