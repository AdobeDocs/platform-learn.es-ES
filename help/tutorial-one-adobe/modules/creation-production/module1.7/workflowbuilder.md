# 1.7 Firefly Creative Production for Enterprise

Firefly Creative Production for Enterprise se centra principalmente en los tecnólogos creativos y los creadores de soluciones que crean y configuran flujos de trabajo.

Con Firefly Creative Production for Enterprise, puede:

- Generar flujos de trabajo: cree flujos de trabajo de producción de varios pasos mediante una interfaz visual con asistencia manual o de IA.
- Ejecutar flujos de trabajo: ejecute flujos de trabajo a escala, incluido el procesamiento por lotes, con un tiempo de ejecución sólido que funcione fuera del explorador.
- Implementar flujos de trabajo: Publique flujos de trabajo como extremos de API o IU programáticas para su integración en otras herramientas y sistemas.
- Integración de flujos de trabajo: conecte flujos de trabajo a sistemas empresariales como DAM, flujos de activación y cadenas de suministro de contenido.

Los casos de uso principales admitidos son estos:

- Adaptaciones del canal de medios sociales, como la creación de variantes de tamaño para cada plataforma y tipo de medios sociales.
- Preparación de recursos, como la preparación de recursos para utilizarlos en superficies adicionales como impresión o web.
- Retargeting de contenido, como actualizar y reutilizar elementos visuales para volver a atraer audiencias interesadas.
- Comercio electrónico, como la normalización de imágenes de productos y la creación de variantes específicas de segmentos para distintas audiencias.
