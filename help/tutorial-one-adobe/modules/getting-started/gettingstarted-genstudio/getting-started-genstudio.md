# Primeros pasos: GenStudio

En este módulo configurará todo para que su entorno y equipo estén listos para empezar a tomar los laboratorios técnicos de GenStudio.

## Objetivos de aprendizaje

- Obtenga información sobre cómo configurar la extensión de Chrome
- Configurar el proyecto de Adobe I/O y Postman o Postbuster
- Instale las aplicaciones necesarias para tomar GenStudio Tech Labs

En este ejercicio instalará y configurará la extensión de Chrome para poder utilizarla y personalizar el contenido del tutorial.

[Configuración del proyecto de Adobe I/O](./ex2.md)

En este ejercicio configurará el proyecto de Adobe I/O que se utilizará para interactuar con varios extremos de API de Adobe.

[Opción 1: configuración de Postman](./ex3.md)
