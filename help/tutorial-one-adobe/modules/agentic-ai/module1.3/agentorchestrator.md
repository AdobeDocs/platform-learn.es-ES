# 1.1 Agent Orchestrator

## Objetivos de aprendizaje

Aprenda a utilizar los agentes de Adobe Experience Platform y Agent Orchestrator para lo siguiente:

- Analizar tendencias de compra
- Identificación de audiencias de alta tendencia
- Validar el rendimiento del recorrido
- Cree un nuevo recorrido para el despliegue de CitiSignal Fiber Max

## Requisitos previos

- Acceso a Adobe Journey Optimizer y Customer Journey Analytics
- Familiaridad con [conceptos de Agent Orchestrator](https://experienceleague.adobe.com/es/docs/experience-cloud-ai/experience-cloud-ai/agents/agent-orchestrator)
- Acceso a los datos de demostración de CitiSignal: compras, pedidos, audiencias y Recorridos
